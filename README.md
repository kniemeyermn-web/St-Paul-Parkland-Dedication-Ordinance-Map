# St. Paul Parkland Dedication Map

A tool to visualize parkland dedication fee generation and usage on park projects in St. Paul, MN.

### Key Features
* **Fee Visualization:** See where projects resulted in the collection of Parkland Dedication fees.
* **Funding Calculator:** Click a park from the sidebar or any spot on the map to see the total available funding within a **0.5-mile radius**.
* **Address Search:** Use the sidebar to jump to specific project locations.

### Data Source
Data was provided by St. Paul Parks and Recreation staff on April 15th, 2026. This tool is intended for advocacy and research.
